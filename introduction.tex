\chapter{Introduction}


\paragraph{}
Ce cours a pour but de présenter l'architecture et le système de production du noyau Linux, afin
d'avoir les bases nécessaires pour l'implémentation de modules noyau.\\
Afin de permettre aux élèves de facilement s'orienter dans l'architecture d'un noyau monolithique
tel que Linux, ce cours propose en premier lieu un historique des noyaux et décrit les différentes
architectures existantes.

\paragraph{}
Afin de bien comprendre les arcanes du noyau Linux et de son système de production, il est conseillé
d'avoir de bonnes notions de langage C et être habitué aux systèmes de Makefile. Le cours sur les
outils GNU est un bon point de départ pour se former à ces outils de production. Il est considéré
dans ce cours que les élèves possèdent déjà ces connaissances.
